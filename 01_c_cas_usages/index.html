Cas d'usages

Inscription et Connexion Utilisateur

Description : Un utilisateur peut s'inscrire, se connecter et gérer son compte.

sequenceDiagram participant User as Utilisateur participant Auth as Service d'Authentification User->>Auth: Inscription/Connexion Auth->>User: Confirmation/Erreur

Consultation du Catalogue de Produits

Description : Un utilisateur peut consulter le catalogue de produits disponibles.

sequenceDiagram participant User as Utilisateur participant Catalog as Service Catalogue User->>Catalog: Requête de Catalogue Catalog->>User: Liste de Produits

Processus de Paiement

Description : Un utilisateur peut passer au processus de paiement pour finaliser son achat.

sequenceDiagram participant User as Utilisateur participant Payment as Service Paiement User->>Payment: Initiation du Paiement Payment->>User: État du Paiement (Success/Erreur)

Gestion des Subventions

Description : Un utilisateur peut utiliser des subventions pour payer une partie de son achat.

sequenceDiagram participant User as Utilisateur participant Subsidy as Service Subventions User->>Subsidy: Utilisation de Subventions Subsidy->>User: Confirmation/Erreur
